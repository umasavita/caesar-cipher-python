# 🔐 Caesar Cipher in Python

## 📌 Description
This project implements the Caesar Cipher algorithm for encrypting and decrypting text using Python.

## 🛠 Tools Used
- Python

## 🚀 Features
- Encrypt text using shift value
- Decrypt text back to original
- Handles uppercase and lowercase letters

## ▶️ How to Run
python caesar_cipher.py


## 📸 Output Example
Input: hello  
Shift: 3  
Output: khoor
